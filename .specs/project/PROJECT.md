# Atropelis Simulator

## Vision
A GTA 1-inspired top-down browser game where players navigate a city in a white car, aiming to score points by hitting pedestrians and light vehicles while avoiding collisions with other cars.

## Key Goals
1.  **Top-Down Driving Mechanics:** Smooth, GTA 1-style car control (keyboard and touch).
2.  **Dynamic City Environment:** Procedural or static city layout with AI-controlled pedestrians, bikes, motorcycles, and cars.
3.  **Scoring & Progression:** Point-based system (+1 Person, +2 Bike, +5 Motorcycle, -5 Car) with a 1-minute timer.
4.  **Persistent Leaderboard:** Local Storage integration for tracking nicknames and high scores (with dates).
5.  **Cross-Platform Responsive UI:** Seamless experience across desktop and mobile browsers.

## Target Stack
- **Frontend:** React (TypeScript) + Vanilla CSS (for maximum flexibility and performance).
- **Game Engine:** Canvas API (for the top-down 2D rendering).
- **State Management:** React Context/Hooks + Local Storage for persistence.
